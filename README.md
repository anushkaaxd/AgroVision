## 🌿 About AgroVision

AgroVision is an AI based plant disease detection system that helps identify plant health issues using image processing and deep learning

It aims to support farmers and researchers by providing fast and accurate predictions

---

## ✨ Features

🌸 Upload plant image
🌸 Detect disease instantly
🌸 Clean UI
🌸 AI powered predictions

## ⚙️ Installation

```bash
git clone https://github.com/anushkaaxd/AgroVision.git
cd AgroVision
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

---

## ▶️ Run

```bash
python app.py
```
